# manageZoo
Api en Node JS s'occupant de la gestion d'un zoo

##Prérequis

``npm``

##Installation

```npm install```

##Taches

- connexions / inscription du user | OK
- empolyee : mettre en maintenance un espace (admin seulement) | OK
- employee : mettre le mois pour réparer le space | OK
- ajouter des animaux dans les espaces (type employee : patron zoo) | OK
- ajouter une info dans le carnet d'animaux (type d'employee : veterinaire)
- vérifier si le zoo peut ouvrir (au moins 1 personne à
  l'accueil, 1 soigneur, 1 agent d’entretien ainsi qu’ 1 vendeur.
  ) | OK
- création des billets(plusieurs type dont le PASS Escape game) (patron zoo)
- contrôle d'accés des user (si son billets lui permet d'entrer)
- Des statistiques quotidiennes et hebdomadaires devront permettre de mettre en évidence l’affluence de
  notre magnifique zoo par espace
  
- Taux de fréquentation du zoo et des espaces en temps reel

- Ouverture nocturne du zoo avec PASS Night (par un admin)
-  Afin d'accéder à l'API il faudra avoir un compte, seul un employé peut faire des actions pour le parc. | OK

# manageZoo
Api en Node JS s'occupant de la gestion d'un zoo

##Prérequis

- npm
- Apache serveur
- Mysql serveur
- NodeJS


##Installation

```npm install```


Créer un fichier```.env``` à la racine du projet
Exemple : 
```
PORT=3000
DB_HOST=DATABASE_IP
DB_DRIVER=mysql
DB_NAME=YOUR_DATABASES_NAME
DB_USER=YOUR_USERNAME
DB_PASSWORD=YOUR_PASSWORD
DB_PORT=DATABASES_PORT
```
